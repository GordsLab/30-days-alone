# 30 Days Alone

## Game Purpose

In this Python text-based game, you experience the life of someone lost in a forest 🌲.
For 30 days, you must survive by facing monsters, living through unforgettable adventures, and managing your health ❤️ and hunger 🍖.

## Game Development

I started building this game because I wanted to challenge myself and create my first large Python project. I spent a lot of time making the gameplay as enjoyable as possible.
This project marks the beginning of my programming journey and is the biggest Python project I have created so far. In the future, I plan to develop a 2D version of 30 Days Alone after learning the Pygame library.
I also enjoyed fixing bugs, designing the story, and translating the entire game from French into English so that international players could enjoy the experience.

## How to Run

### Requirements
- Python 3.12 or higher
- Colorama

### Install Colorama
Open a terminal in the project folder and run:

```

pip install colorama

```

### Run the game
Start the game with:

```

python 30\_days\_alone.py

```

Enjoy the game 🙂
